## Description
This project features a dashboard in Excel designed to make data analysis easy and intuitive. It provides visualizations and tools to analyze sales data efficiently.

## Requirements
- Microsoft Excel (any recent version)

## Setup
1. Download and open the `Sales_Analysis.xlsx` file in Microsoft Excel.
2. Ensure macros are enabled in Excel for full functionality.

## Usage
1. Open the Excel file.
2. To use macros:
   - Enable macro recording if needed.
   - Record and apply the necessary macros as per your analysis requirements.
3. Use the dashboard to explore and analyze the sales data.

## Data Sources
The sales data can be obtained from Kaggle. Import or load your data into the Excel file as needed.
